☁️ Weather-App-Sri-Lanka- ☁️

A soft, kawaii-inspired weather application designed specifically for finding weather updates in Sri Lanka 🇱🇰. 
Built with a dreamy pastel aesthetic, frosted glass effects, and a friendly "cloud-like" interface.

🎀 Features

Localized Search: Automatically filters results to cities within Sri Lanka (using ISO code LK).

Kawaii UI: Soft pastel gradients, rounded "pill" buttons, and dashed borders for a scrapbooking feel.

Floating Animations: The weather card gently floats to give a dreamy, cloud-like experience.

Live Data: Fetches real-time temperature and weather conditions via the OpenWeatherMap API.

Metric by Default: Displays temperature in Celsius (°C) without manual conversion.

🛠️ Tech Stack

HTML5 - Semantic structure.

CSS3 - Custom animations, glassmorphism, and pastel color palette.

JavaScript (ES6+) - Asynchronous fetch calls and DOM manipulation.

API - OpenWeatherMap API.

🎨 Design Elements

  Font: Clean, rounded sans-serif stack for readability.

  Colors:  #fce4ec (Sakura Pink)

          #e1f5fe (Clear Sky Blue)

          #ff8fb1 (Accent Pink)

  Icons: 
  Dynamic weather icons from OpenWeatherMap rendered with a @2x scale for extra cuteness.
